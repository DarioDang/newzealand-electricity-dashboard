New Zealand Electricity Dashboard — Live Grid Data

This dashboard provides real-time New Zealand electricity market data including wholesale electricity prices ($/MWh) across 14 grid zones: Auckland (OTA), Wellington (HAY), Christchurch (ISL), Benmore (BEN), Waikato, Huntly, and more. Data is sourced from the em6 API and refreshed every 30 minutes via an automated GitHub Actions pipeline backed by Neon Postgres and dbt.

Key metrics tracked: renewable generation percentage, carbon intensity in gCO₂/kWh, North Island and South Island price spread, fast instantaneous reserve (FIR) and slow instantaneous reserve (SIR) prices, and 30-day price trends.

Built with FastAPI, Plotly, and vanilla JavaScript. Open source on GitHub. Ideal for anyone researching New Zealand electricity prices, carbon emissions, or renewable energy generation data.
